# Project Title

Schedule Automation Using Python

## Description

Project ini adalah implementasi sederhana dari schedule automation menggunakan Python. Proyek ini memungkinkan pengguna untuk menjadwalkan dan menjalankan tugas-tugas tertentu pada waktu yang ditentukan secara otomatis.
Pada script ini bisa dilakukan pengembangan dan kustom menggunakan command dari beberapa produk yang kalian inginkan untuk menjalankan fungsi schedule ini.

## Features

- Menjalankan tugas sesuai jadwal
- Log hasil eksekusi tugas

## Requirements

- Python 3.7 atau versi lebih baru
- Modul-modul Python: `schedule`, `time`, `datetime`
- Install Makefile untuk menagement project :
  1. Copy paste to your powersheel from administrator Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
  2. Install make on your cmd from administrator "choco install make"
